# 端午出行

出行日期：6.8 ~ 6.10

温州天气：多云?

出行轨迹：6.7晚出发到南麂岛，6.8晚出发到雁荡山脚，6.10晚回家

6.7晚：杭州站(G7615 18:15  r138)  =>  温州南（G7339 21:45）   =>  平阳站

6.8早10.30 ~ 18.30 : 南麂岛  

6.8晚（或者6.9早）：平阳站 => 雁荡山站（R43）  （或者晚上打车~R300）

6.10晚：雁荡山站  =>  杭州站（喵的，票一下就没了，打车回杭州700+）

住宿：很贵

## 雁荡山

地图：

大龙湫：?50元，下雨后大龙湫很美。

方洞：?40元，有铁索桥、方洞栈道和咖啡馆，看群山风景很好。

灵岩：?50元，小的玻璃栈道、小龙湫、断肠崖、灵岩寺在这里。

灵岩-方洞：景区联票?80元，2024元旦新开通的卧云栈道。要看小龙湫和断肠崖要从方洞往灵岩走（下坡路相对好走）。

灵峰：?45元，游览大约2小时（不包含往返真际寺）。果盒桥、合掌峰、观音洞、北斗洞和许多奇峰怪石。在东西瑶台上可以看到嵌在山体内的北斗洞。

三折瀑景区：?20元，下折瀑、中折瀑游览大约1小时。去的时候是枯水季下折瀑和中折瀑没什么水，上折瀑需要往返3小时很听劝没有去。上折瀑的终点通往森林公园（净名谷）需要单独另外买票。：?50元（下雨才美）
