## 出行

6.8早：4~5人打车到七里杨帆，滴滴预约~R200
6.8晚：住宿在严城古镇
6.9晚：住宿在新安江（或千岛湖）

七里杨帆（80羊毛，富春山居图再现）    =>   时间松弛去玉泉寺     =>   严城古镇（80羊毛，杭州的大理古城）   => 双桂苑民宿（距离近，比较实惠）
=>     大慈岩(68羊毛，佛教文化)     =>   灵栖洞（80羊毛）   =>时间松弛去千岛湖石林（号称华东第一石林，但看评论说不好看）  =>     新安江（110羊毛，乘船赏梦幻夜景） 
=>   千岛湖（理想花园）

# 出发前准备

登山杖、钉鞋、头灯（看日出要走会夜路）、自热米饭（山上有自助午餐但很贵）

上山后，在山上住，看日落日出（日出时间5.30到6.00, 要早起占位置：光明顶，丹霞峰，鳌鱼峰），

住宿（招待所、帐篷、白云宾馆多人经济房）
