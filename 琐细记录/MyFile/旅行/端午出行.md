# 雁荡山+南麂岛


天气多云：? （-6° ~ 2°）

# 线上购票

南麂客运中心：票很紧急

# 杭州到黄山出行方案

高铁：杭州东站（西站也行） => 黄山北站

- 方案一：
周五晚上到黄山下，早上去莲花峰排队
- 方案二：
周六上午从杭州出发，可以下午去莲花峰

然后周末下午回

# 出发前准备

登山杖、钉鞋、头灯（看日出要走会夜路）、自热米饭（山上有自助午餐但很贵）

上山后，在山上住，看日落日出（日出时间5.30到6.00, 要早起占位置：光明顶，丹霞峰，鳌鱼峰），

住宿（招待所、帐篷、白云宾馆多人经济房）

# 旅游路线


黄山北站下车  =>  黄山客运中心（`30元`）  =>  黄山风景区南大门   =>   （汤口）换乘中心做大巴索道口(可以先吃个饭再坐索道)

上山后，天气好适合去莲花峰：

迎客松  => 莲花峰（下午可能人少点，排队没那么长，或者赶早去）   
=> 百步云梯   =>   光明顶   => 飞来石    =>    排云亭     =>    
丹霞峰    =>     猴子观海   =>    梦笔生花    =>   始信峰    =>  索道下山（索道运行时间：7.00到17.00）
