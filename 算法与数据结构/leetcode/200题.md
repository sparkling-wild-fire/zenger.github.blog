# 有意义的200题

## 无重复字符的最长子串

我的思路：每次将指针向右移1，并计算以当前指针为终点的最长子串长度，但这种移动方法无法利用哈希（唯一性=>哈希 ）

实现：
```C++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        /*
            记录以每个字符结尾的无重复字串长度，取最大值
        */
        
        int slen=s.size();
        if(slen==0){return 0;}   // Tip：空串，返回0；写后面代码的时候感觉不好统一空字符串，就先处理了
        int zmax=1;
        int idx_len=1;
        int idx=1;
        bool flag;   // 判断子串遍历过程中是否重复
        while(idx<slen){
            int i;
            flag=false;      
            for(i=idx-idx_len;i<idx;++i){   // 查找当前字符最近重复的位置abca
                if(s[i]==s[idx]){
                    flag=true;
                    break;
                }     
            }
            
            idx_len = flag ? idx - i : ++idx_len;
            if(zmax<idx_len){
                zmax=idx_len;
            }
            ++idx;
        }
        return zmax;
    }
};
```

更好的思路：哈希+滑动窗口(计算以当前指针为`起点`的最长子串)

实现：
```C++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        // 哈希集合，记录每个字符是否出现过
        unordered_set<char> occ;
        int n = s.size();
        // 右指针，初始值为 -1，相当于我们在字符串的左边界的左侧，还没有开始移动
        int rk = -1, ans = 0;
        // 枚举左指针的位置，初始值隐性地表示为 -1
        for (int i = 0; i < n; ++i) {
            if (i != 0) {
                // 左指针向右移动一格，移除一个字符
                occ.erase(s[i - 1]);
            }
            while (rk + 1 < n && !occ.count(s[rk + 1])) {
                // 不断地移动右指针
                occ.insert(s[rk + 1]);
                ++rk;
            }
            // 第 i 到 rk 个字符是一个极长的无重复字符子串  => 抓住这里，哈希记录的是一个无重复的滑动窗口
            ans = max(ans, rk - i + 1);
        }
        return ans;
    }
};
```
