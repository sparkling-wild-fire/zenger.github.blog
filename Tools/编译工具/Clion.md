# Clion

## 基础设置

1. 打开项目时，选择是新窗口还是本窗口：设置=>系统设置

2. CLion配置SVN，博客用的git，algoserver的用svn，分别配置。[参考链接](https://blog.csdn.net/cmhdl521/article/details/127775960)

3. 快捷键：
   - alt+shift: 多行编辑
   - ctrl+G:跳转到指定行：列  => 下次百度不到了就百度vscode的
